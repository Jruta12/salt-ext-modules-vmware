Welcome to Salt Extension Modules for VMWare
============================================

This project is a collection of extension modules for the `Salt Project`_.

.. _Salt Project: https://docs.saltproject.io

It's currently under active development, and is split into two main areas: NSX
and the rest of vSphere.
